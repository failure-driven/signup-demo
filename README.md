# Signup Demo

Demo Behaviour Driven Development (BDD) app.

## getting started

run all the tests

```
bin/full-build
```

## branches

Jump to form not redirecting branch

```
make ddd1
```

Jump to no react branch

```
make ddd2
```

Jump to demo OmniAuth branch

```
make oauth
```

Jump back to master

```
make master
```
